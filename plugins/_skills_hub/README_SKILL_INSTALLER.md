# Skill Installer Plugin

This plugin allows you to browse, install, and update skills from [agentskill.sh](https://agentskill.sh) directly within Nova.

## Features

- Browse trending, new, and popular skills
- Install skills globally or for specific projects/agents
- Update installed skills
- Install skills from Git repositories
- Install skills from ZIP files

## Installation

1. Place this plugin in `/a0/plugins/_skill_installer/`
2. Restart Nova

## Usage

1. Open the Skill Installer from the Plugins menu
2. Browse available skills or use the Git/ZIP tabs to install from other sources
3. Click on a skill to install it

## Configuration

No additional configuration is required.

## Dependencies

- Node.js 18+
- @agentskill.sh/cli (installed automatically when needed)
